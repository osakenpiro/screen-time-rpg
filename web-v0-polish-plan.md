# 画面録 Web版 磨き込みプラン v0（無料導線・プライバシー100%）

> 位置づけ：売れる版（ネイティブ）とは**別物**。Web版は **無料の入口・コンセプト実証・マーケ素材**。
> 強み：①手入力でも「あなたの六角形は？」が成立 ②**データが端末から出ない＝プライバシー100%** ③反・煽り。

## P0（効く・軽い／最優先）
1. **シェアを“絵”で出す**：hexagon＋調和＋称号を `canvas` で PNG 化 → Web Share API／ダウンロード。
   いまのテキストシェアより圧倒的に拡散する（SNS＝画像）。**導線の核**。工数=中。
2. **ビルド称号**：調和×状態異常×主軸から称号を生成（例「均衡の賢者」「技の鬼」「呪われた発信者」「探究の旅人」）。
   RPGらしさ＋シェア動機。工数=小。
3. **プライバシー信頼シグナルを前面に**：ヒーロー直下に常設バッジ「🔒 入力は端末のブラウザにだけ・送信ゼロ」。
   安心して入力させる＝CV向上。工数=小。

## P1
4. **オンボーディング1画面**：初回だけ「読み方（調和が主役／Lvは副次／呪い＝過剰）」を3カードで。工数=小。
5. **プリセット拡充**：今の“サンプル”を「軽め／重め／SNS過多型／開発者型」など複数に。比較で読み方が伝わる。工数=小。
6. **モバイル微調整**：入力欄のタップ性、hexラベルの折返し、フォントサイズ。工数=小。
7. **週次入力の摩擦減**：前回値プリフィル（localStorage済）＋「先週との差」を**矢印でなく六角形の重ね描き**で（煽らない差分）。工数=中。

## P2
8. **ネイティブ版への導線＝需要計測**：フッターに「自動版を待つ（メール登録/告知リンク）」。作る前に欲しい人を数える。工数=小。
9. **OGP画像**：シェア時カード絵（六角形モチーフ）。工数=小。
10. **自ツール相互リンク**：furiko / yutori / bakuun / mitsuami と相互導線（#全人類UX改善 の回遊）。工数=小。
